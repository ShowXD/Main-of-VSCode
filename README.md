### HTML CSS Support
* 使 VSCode 支援 html 和 css。

### HTML Snippets
* 完整的 html5 標籤。

### Mithril Emmet
* 使用簡短的語法產生繁複的內容

### open in browser
* 能夠直接滑鼠右鍵點擊預覽頁面。

### Python
* 使其支援Python語法

### Code Runner
* 支援各種語言在上面執行

### SFTP
* 使其譨購遠端上傳伺服器檔案
#### 操作方法
1. 按下F1，輸入SFTP
2. 選擇SFTP configure，就會生成設定檔

* 預覽網頁時以本地端server方式開啟
